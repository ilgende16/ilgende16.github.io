Individual JSP Program

My individual JSP program for this project is a CRUD database for the top NFL single game passing records. The source code for this program can be found in my GitHub repository here.

The home page. Begin by creating the quarterback database.

After creating the quarterback database, you can add a new record using the create page. Although there are only limited fields currently, this program is easily expanded to include any information desired.

The create page. I coded in the first 5 entries of the database, so I am adding the 6th.

Once you add all of the entries you want, you can view the current database entries by retrieving the records.

The retrieve page. For the sake of updating an entry, I purposely made entry 3 incorrect.

If there are any entires with incorrect information, you can edit them using "update quarterback record". In this case, I am going to edit entry 3, as I purposely spelled his last name incorrectly and have the wrong team entered.

The update page. Here I have corrected the spelling of his last name and entered the right team.

The final process you can use for the database is to delete a record. In a database like this, it wouldn't be used too frequently. But, if you accidentally add an extra record or if somebody has a game taken away for PEDs, this would come in handy.

The delete page. Sorry Boomer, nothing personal.

Team JSP Program

Our team JSP program for this project is a CRUD database for the books in Elliott 216. The source code for this program can be found in my GitHub repository here.

The home page. Begin by creating the book database.

After creating the book database, you can add a new book using the create page. Although there are only limited fields currently, this program is easily expanded to include any information desired.
